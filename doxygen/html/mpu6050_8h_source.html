102 /* Register PWR_MGMT_1 0x6B
103 
104 Description:
105  This register allows the user to configure the power mode and clock source.
106  It also provides a bit for resetting the entire device, and a bit for disabling
107  the temperature sensor.
108 
109  By setting SLEEP to 1, the MPU-60X0 can be put into low power sleep mode.
110 
111  When CYCLE is set to 1 while SLEEP is disabled, the MPU-60X0
112  will be put into Cycle Mode.
113 
114  In Cycle Mode, the device cycles between sleep mode and waking up to take
115  a single sample of data from accelerometer at a rate determined by LP_WAKE_CTRL.
116 
117  To configure the wake frequency, use LP_WAKE_CTRL within the Power Management 2
118  register (Register 108).
119 
120 +--------------+-------+-------+------+----------+--------+--------+--------+
121 | Bit7 | Bit6 | Bit5 | Bit4 | Bit3 | Bit2 | Bit1 | Bit0 |
122 +--------------+-------+-------+------+----------+--------+--------+--------+
123 | DEVICE_RESET | SLEEP | CYCLE | - | TEMP_DIS | CLKSEL[2:0] |
124 +--------------+-------+-------+------+----------+------------+------+------+
125 
126 
127 +--------+------------------------------------------------+
128 | CLKSEL | Clock Source |
129 +--------+------------------------------------------------+
130 | 0 | Internal 8MHz oscillator |
131 | 1 | PLL with X axis gyroscope reference |
132 | 2 | PLL with Y axis gyroscope reference |
133 | 3 | PLL with Z axis gyroscope reference |
134 | 4 | PLL with external 32.768kHz reference |
135 | 5 | PLL with external 19.2MHz reference |
136 | 6 | Reserved |
137 | 7 | Stops the clock and keeps the timing generator |
138 | | in reset |
139 +--------+------------------------------------------------+
140 
141 
142 
143 Parameters:
144 
145 DEVICE_RESET:
146  When set to 1, this bit resets all internal registers to their default values.
147  The bit automatically clears to 0 once the reset is done.
148  The default values for each register can be found in Section 3.
149 SLEEP:
150  When set to 1, this bit puts the MPU-60X0 into sleep mode.
151 CYCLE:
152  When this bit is set to 1 and SLEEP is disabled, the MPU-60X0 will cycle
153  between sleep mode and waking up to take a single sample of data from
154  active sensors at a rate determined by LP_WAKE_CTRL (register 108).
155 TEMP_DIS:
156  When set to 1, this bit disables the temperature sensor.
157 CLKSEL:
158  3-bit unsigned value. Specifies the clock source of the device.
159 */
